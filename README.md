# Vanilla Arch Linux Setup — KDE Plasma + Hyprland

Dual-environment setup on vanilla Arch Linux. Both sessions available at SDDM login.
Theme: **Catppuccin Mocha** (dark) applied consistently across both environments.

---

## System State at Start

| Item | Detail |
|------|--------|
| OS | Arch Linux (rolling) |
| Display manager | SDDM (already running) |
| KDE Plasma | 6.6.4 (installed, unconfigured) |
| Hyprland | 0.54.3 (installed, autogenerated config only) |
| Sessions at SDDM | `plasma.desktop`, `hyprland.desktop`, `hyprland-uwsm.desktop` |
| User | `mustafa` (wheel group) |

## How to Use

### Run setup
```bash
bash ~/setup-desktop.sh
```

### Log in
Log out → SDDM login screen → choose session:
- **Plasma** → KDE Plasma
- **Hyprland (UWSM)** → Hyprland

### Add a new user later
```bash
sudo bash ~/new-user.sh <username>
```
Copies all configs to the new user's home directory.

### Change wallpaper (Hyprland)
Edit `~/.config/hypr/autostart.conf` — update the `swaybg` path.

### Change monitor scale (Hyprland)
Edit `~/.config/hypr/monitors.conf` — uncomment the preset matching your display.
